# Minesweeper

# C언어로 구현한 보물 찾기 게임입니다.

모든 입력은 'R' or 'r' , 'Q' or 'q' , 방향키(↑,↓,←,→), Enter 키만 가능합니다. 

메인 화면에서 Easy, Normal, Hard 세 가지의 난이도를 고를 수 있습니다.
